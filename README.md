# Portfólio — Maria Luisa Sanches

**[→ Acesse em maria-luisa-sanches.vercel.app](https://maria-luisa-sanches.vercel.app)**

[![Assistir demo do portfólio](https://img.youtube.com/vi/SEU_VIDEO_ID/maxresdefault.jpg)](https://youtu.be/wjlN1A8cL4w)

Portfólio pessoal desenvolvido com foco em identidade visual autoral e experiência de navegação cinematográfica. A estética combina paleta vinho escuro com elementos clássicos e interfaces inspiradas em sistemas operacionais e terminais.

---

## Seções

- **Hero** — apresentação com texto rotativo animado e modelo 3D interativo via Spline
- **Terminal** — listagem de projetos estilizada como terminal bash
- **Projetos** — cards com preview, descrição e stack de cada projeto
- **Conquistas** — galeria com marcos acadêmicos e profissionais em formato de quadros
- **Skills** — tecnologias organizadas por categoria
- **Sobre** — interface estilo desktop com bio e foto
- **Contato** — cartão postal com links para LinkedIn, GitHub e e-mail

---

## Stack

- **React 19** + **Vite**
- **Framer Motion** — animações e transições entre seções
- **Spline** (`@splinetool/react-spline`) — modelo 3D interativo no Hero
- **React Router DOM** — navegação
- **CSS puro** — estilização manual com variáveis e design system próprio
- **Deploy:** Vercel

## Como rodar localmente

```bash
# Clone o repositório
git clone https://github.com/marialuisasanches/maria-luisa-sanches.git

# Entre na pasta
cd maria-luisa-sanches

# Instale as dependências
npm install

# Rode em desenvolvimento
npm run dev
```

Acesse `http://localhost:5173` no navegador.
